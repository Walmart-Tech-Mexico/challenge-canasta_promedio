# Canasta promedio

**Nivel:** Intermedio

## Descripción
Calcular el ticket promedio por cliente a partir de cientos de compras.

## Objetivo
Implementa una solución en Java que cumpla con la lógica descrita. Usa la plantilla `Main.java` para comenzar.

## Cómo empezar
1. Clona este repositorio.
2. Dirígete a la carpeta `Canasta_promedio`.
3. Abre `Main.java` y escribe tu solución.
4. ¡Comparte tu solución con la comunidad!
